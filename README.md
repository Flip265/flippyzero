# Flippyzero

A comprehensive Roblox debugging and inspection tool created by Flip265.

## Description

Flippyzero is a Roblox Lua script that provides an advanced GUI-based interface for inspecting and analyzing Roblox games. It includes multiple modules for spying on scripts, HTTP requests, memory, and more.

## Features

- **RSpy** - Remote function/event inspection
- **ESpy** - Event analysis
- **HTTPSpy** - HTTP request monitoring
- **ScriptScanner** - Script detection and analysis
- **MemoryScanner** - Memory inspection tools
- **CodeBox** - Code visualization
- **Settings** - Configurable options
- **Minimalist UI** - Clean, draggable window interface

## Installation

```lua
loadstring(game:HttpGet("hhttps://raw.githubusercontent.com/Flip265/flippyzero/refs/heads/main/flippyzero.lua"))()
```

## Usage

Once loaded, the Flippyzero GUI will appear as a draggable window. Use the following features:

- **Minimize/Close Buttons** - Control the window state
- **Navigation Buttons** - Switch between different inspection modules
- **Settings** - Customize tool behavior
- **Home** - Main dashboard

## Components

- `UIControl` - Main UI controller
- `Settings` - Configuration module
- `Shared` - Shared utilities
- `CodeBox` - Code display and formatting
- `ToString` - Object to string conversion
- `HookFunction` - Function hooking utilities
- `Changelogs` - Version history
